## 📝 Description

This repository contains C programs demonstrating the applications of **Stack** and **Queue** data structures.

The programs are menu-driven and implemented using arrays to simulate real-world scenarios such as expression checking and queue management.

---

## 📂 Programs Included

### 🔹 Q1: Reverse a String using Stack

* Uses stack (LIFO) to reverse a given string

### 🔹 Q2: Balanced Parentheses

* Checks whether an expression has balanced parentheses using stack

### 🔹 Q3: Next Greater Element

* Finds the next greater element for each array element using stack

### 🔹 Q4: Printer Queue Simulation

* Simulates printer job scheduling using queue (FIFO)

### 🔹 Q5: Circular Queue Operations

* Implements circular queue using array
* Supports:

  * Enqueue
  * Dequeue
  * Peek
  * Display
